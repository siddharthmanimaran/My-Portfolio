# Siddharth Manimaran — Portfolio

A personal portfolio website built with [Next.js](https://nextjs.org/), [TypeScript](https://www.typescriptlang.org/), and [Tailwind CSS](https://tailwindcss.com/).

## Features

- **About** — Introduction and skills overview (Frontend, Backend, Database)
- **Experience** — Professional work history
- **Resume** — Downloadable PDF resume
- **Dark / Light mode** toggle
- Responsive layout

## Tech Stack

- [Next.js](https://nextjs.org/) 12
- [React](https://reactjs.org/) 17
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Icons](https://react-icons.github.io/react-icons/)

## Getting Started

Install dependencies:

```bash
npm install
# or
yarn
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the portfolio.

## Build

```bash
npm run build
npm run start
```

## Lint

```bash
npm run lint
```

## Deploy

The easiest way to deploy this app is with the [Vercel Platform](https://vercel.com/). Check out the [Next.js deployment documentation](https://nextjs.org/docs/deployment) for more details.
